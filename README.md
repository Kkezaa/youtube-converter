# YouTube Converter

A Python application to download YouTube videos as MP4 or extract audio as MP3, using a simple Tkinter graphical interface.

## Features
- Download YouTube videos as **MP4**
- Extract audio as **MP3**
- Easy-to-use **Tkinter GUI**
- Uses `pytubefix` for reliable downloads

## Requirements
- Python 3.x  
- `pytubefix`  
- Tkinter (included with most Python installations)

> If Tkinter is missing on Linux:  
```bash
sudo apt-get install python3-tk
```

## Installation
```bash
# Clone the repository
git clone https://github.com/kkezaa/youtube-converter.git
cd youtube-converter

# Install dependencies
pip install pytubefix
```

## Usage
```bash
python main.py
```

A window will open where you can paste a YouTube URL, choose MP4 or MP3, and download the content.
